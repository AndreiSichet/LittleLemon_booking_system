# Little Lemon – Restaurant Booking System

## Description

Little Lemon is a restaurant booking system built with Django 4.1. This project was completed as part of the Coursera Backend Professional Certificate program. The application allows customers to view the menu, see details for each dish, and make online table reservations. It also manages reserved time slots and displays existing bookings for each day.

The system includes:

- Home page (/) with special offers and quick links.
- About page (/about/) with information about the restaurant and owners.
- Menu page (/menu/) listing all dishes and their prices.
- Menu item detail page (/menu/<id>/) showing description and image.
- Book a table page (/book/) to make reservations.
- Reservations page (/reservations/) showing bookings for the selected date.

## Features

- View the menu and dish details.
- Make reservations with dynamically updated available time slots.
- Real-time updates using JavaScript and fetch API.
- Reservation management through a MySQL database (or SQLite for testing).
- Responsive, user-friendly interface.
